7.12. RESET SESSION

Synopsis

RESET SESSION name
RESET SESSION catalog.name

Description

Reset a session property value to the default value.

Examples

RESET SESSION optimize_hash_generation;
RESET SESSION hive.optimized_reader_enabled;
